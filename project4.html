DNN from scratch

Description du projet

Ce projet a été réalisé dans le cadre académique de Télécom SudParis. L'objectif de ce projet a été de développer un Deep Neural Network sur Numpy afin qu'il puisse classifier les chiffres provenant de la base de données MNIST. Le projet se divise en trois étapes d'implémentation : le RBM (Restricted Boltzmann Machine), le DBN (Deep Belief Network) et pour finir le DNN (Deep Neural Network).

Le code est disponible sur GitHub à l'aide du lien ci-dessous.

Résultats



Trois graphes correspondant aux résultats du DNN sur MNIST. L'ordonnée correspond au taux d'erreur de classification, c'est-à-dire le pourcentage de mauvaises classifications (mauvais chiffre détecté) par rapport à tous les tests.

Le premier graphe en partant de la gauche étudie l'impact de la profondeur du DNN en conservant les autres paramètres fixes à 200 neurones par couche et 10000 données d'entraînement. Le nombre de couche varie de 2 à 9. Le deuxième graphe se concentre sur le nombre de neurones par couche de 100 à 800 neurones. Encore une fois, les autres paramètres sont fixés à 2 couche et 10000 données d'entraînement. Pour le troisième graphe, on étudie l'impact du nombre de données d'entraînement de 1000 à 60000 pour un DNN à deux couches de 200 neurones. Ces trois figures constituent une première exploration d'optimisation des hyperparamètres du DNN.
